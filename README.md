# SqwizzeyTab

Chrome-расширение для замены новой вкладки. Визуальный менеджер закладок с досками, страницами и drag-and-drop.

## Возможности

- Несколько страниц с досками закладок
- Drag-and-drop досок и ссылок
- Импорт закладок Chrome
- Тёмная и светлая тема, кастомный фон
- Корзина с восстановлением (30 дней)
- Экспорт/импорт JSON

## Установка

1. `chrome://extensions` → включить **Режим разработчика**
2. **Загрузить распакованное расширение** → выбрать папку репозитория

## Стек

Vanilla JS/CSS/HTML, Chrome Extension MV3, без сборщика.
