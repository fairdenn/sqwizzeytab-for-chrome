## Установка

1. Открой `chrome://extensions`.
2. Включи `Режим разработчика`.
3. Нажми `Загрузить распакованное расширение`.
4. Выбери папку `daniil-start-bookmark-manager-v3`.

## Обновление с предыдущих версий

1. Старое состояние с `boards` мигрирует в страницу `Home`.
2. Новая версия использует другой storage prefix `dsp_v3_`, чтобы не ломать старые данные.
3. Если нужно перенести старые данные вручную, используй Export JSON в старой версии и Import JSON в новой.

## Ограничения

1. Share Page / Share Board пока копирует текст + JSON в буфер, без публичной ссылки.
2. Для настоящего шаринга по URL нужен backend.
3. Fetch All Titles зависит от CORS и доступности сайта. Некоторые сайты не дадут прочитать title, потому что интернет - это коллекция запретов с картинками.
4. Локальная картинка фона хранится в `chrome.storage.local`, а не sync.
