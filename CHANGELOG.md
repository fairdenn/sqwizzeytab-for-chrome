# Changelog

## v2.9.73 (current)
- 4 колонки досок всегда, round-robin раскладка
- DOM-based drag-and-drop досок между колонками
- Кнопка «+ Добавить доску» в каждой колонке
- Кнопка страницы: текст по центру, карандаш появляется при hover без прыжков
- Нейтрализованы конфликтующие старые drag-обработчики
